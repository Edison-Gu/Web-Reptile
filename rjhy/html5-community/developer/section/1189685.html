        footer

        HTML <footer> 元素表示最近一个章节内容或者根节点（sectioning root ）元素的页脚。一个页脚通常包含该章节作者、版权数据或者与文档相关的链接等信息。

        内容类别

        流量的内容，明显的内容。

        允许的内容

        流内容，但没有<footer>或<header>后代。

        标记遗漏

        没有，起始和结束标签都是强制性的。

        允许的父母

        任何接受流内容的元素。请注意，<footer>元素不能是<address>，<header>或另一个<footer>元素的后代。

        允许ARIA角色

        组，演示文稿

        DOM界面

        HTML元素

        属性

        该元素仅包含 全局属性。

        使用说明：

        • <footer>元素内的作者信息应包含在<address>元素中。

        • <footer>元素不是章节内容，因此在outline中不能包含新的章节。

        示例

        <footer>
  Some copyright info or perhaps some author
  info for an &lt;article&gt;?
</footer>
